## Description

Sales Radar Dashboard is an interactive and modern web dashboard designed to analyze and visualize sales data. It provides a clear overview of sales performance through key performance indicators (KPIs), charts, and data visualization components.

The dashboard helps users track sales trends, monitor business performance, and gain valuable insights from data through a responsive and user-friendly interface. The project was built using React, Vite, Tailwind CSS, Recharts, and Lucide React.
